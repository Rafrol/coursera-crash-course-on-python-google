## Question 5
Replace the _ placeholder and calculate the Golden ratio: $\frac{1+\sqrt{5}}{2}$

```
ratio = (1 + 5**.5) / 2
print(ratio)
```

Another possible answer:
```
ratio = ((1+(5**(1/2)))/2)
```
