## Question 4
Fill in the correct Python commands to put “This is fun!” onto the screen 5 times.

```

```

```

This is fun!
This is fun!
This is fun!
This is fun!
```

## Question 5
Select the Python code snippet that corresponds to the following Javascript snippet:
```
for (let i = 0; i < 10; i++) {
  console.log(i);
}
```
Answer:
```
for i in range(10):
  print(i)
```
